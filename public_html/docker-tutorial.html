Docker - Tutorial

1.) Create a Docker hub account: https://hub.docker.com/signup
2.) Follow this link to install Docker Desktop: https://docs.docker.com/get-docker/
--- It's also recommended to complete the Get started module (found in the previous link)
3.) Troubleshooting: Open Docker Desktop. If an error occurs, try updating your operating system to the most current version. If this doesn't fix it, you may need to go into your system BIOS and enable virtualization. Step by step guide.

Common Docker Commands

$docker build [options] path/url (of image)
$docker run [options] image [command] [args] --- starts a new container
$docker ps --- lists containers and their ids
$docker stop <container-id>
$docker rm <container-id> --- removes container
$docker-compose up --- builds, recreates, and starts docker containers
$docker-compose down --- stops and removes containers started by compose up
