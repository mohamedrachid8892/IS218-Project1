Git

What is Git?

Git is an open source version control platform, meaning that it keeps track of your current and previous code verions and any changes you make. More information here: https://www.educative.io/edpresso/what-is-git

Installation

1.) Create a github account: https://github.com/
2.) Install git: https://github.com/git-guides/install-git

Common Commands

$git init --- creates a new git repository in current directory
$git status --- lists current branch and commit status info
$git add <filename> --- ("." includes all files)
$git commit -m <message describing commit>
$git checkout (-b) <branch> (-b makes new branch and switches to it)
$git push <remote> <branch> --- pushes changes to github
$git merge <branch> --- merges entered branch to current branch
