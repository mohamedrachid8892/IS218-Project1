Vi

Definition

Vi stands for Unix Visual Editor. It is a CLI (or terminal) command that allows for the editing of files within the command line itself.

Syntax

$vi <filename> --- opens a file in the editor

Editor Commands:

i --- enters insert mode
esc --- exits insert mode
shift + colon --- enters command line mode
w --- writes to the file (updates and saves file)
q --- quits the editor
wq --- writes and quits
