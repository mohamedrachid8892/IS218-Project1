Vi - tutorial

Definition:

Vi stands for Unix Visual Editor. It is a terminal command that allows for the editing of files within the terminal itself.

Syntax:

$vi <filename> --- open file in editor

Editor Commands:

i --- enters insert mode
esc --- exits insert mode
shift + colon --- enters command line mode
q --- quits the editor
w --- writes to the file (updates and saves file)
wq --- writes and quits
