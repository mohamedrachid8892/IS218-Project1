Command Line Interface (CLI)

Common Commands

$pwd --- present working directory (gives full path to present directory)
--- / forward slash means absolute directory path (from root of drive)
--- forward slash mean relative path
$cd <foldername> --- change directory
--- .. makes directory position move back one
------ .. is universal signal for reverse
--- . is the current directory
--- ~ takes you to home directory
$ls --- lists everything inside current directory
--- type a directory after ls to list contents of directory
$mkdir <new folder name> --- creates a new directory
$touch <new file name> --- creates an empty file
$rm <folder/file name> --- remove folder/file
--- $rm -r <directory>
------ -r means recursive (loops through directory to delete whole directory)
--- $rm -f
------ -f is a force delete (can be used on .git repositories, but use with caution)
$mv <oldfilename> <newfilename> --- renames files
$mv <file name> <folder name>/ --- moves file to folder
