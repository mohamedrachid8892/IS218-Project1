Azure Cloud

What is Azure Cloud?

Azure Cloud is a cloud service that allows you to build, run, and manage applications across devices. In this case, Azure Cloud will be hosting the website you are creating.

Installation

1.) Create a container image for deployment to Azure Container Instances: https://docs.microsoft.com/en-us/azure/container-instances/container-instances-tutorial-prepare-app
2.) Create an Azure container registry and push a container image: https://docs.microsoft.com/en-us/azure/container-instances/container-instances-tutorial-prepare-acr
3.) Deploy a container application to Azure Container Instances: https://docs.microsoft.com/en-us/azure/container-instances/container-instances-tutorial-deploy-app
